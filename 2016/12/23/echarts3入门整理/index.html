入门基础

获取ECharts

  • 官网下载，开发环境建议下载源代码版本，包含了常见的错误提示和警告。
  • 在ECharts的Github上下载最新的release版本，解压出来的文件夹里的dist目录里可以找到最新版本的echarts库。
  • 通过npm获取echarts，npm install echarts --save
  • cdn引入，可以在cdnjs，npmcdn或者国内的bootcdn上找到ECharts的最新版本。

引入ECharts

ECharts3开始不再强制使用AMD的方式按需引入，代码里也不再内置AMD加载器。因此引入方式简单了许多，只需要想普通javascript库一样用script标签引入。
